##Классификация математических моделей

1. По степени точности:
    - аналитические
    - статистические
2. По фактору времени:
    - статические
    - динамические
3. По степени абстрактности:
    - имитационные
    - аналитические
    - вычислительные
4. По степени используемых функций:
    - линейные
    - нелинейные
5. По степени учета вероятностных факторов:
    - детерминированные
    - вероятностные
